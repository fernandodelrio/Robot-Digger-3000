You are a robot and need to dig and find a golden nugget. You can perform a limit of moviments and can store a limit of instructions denoted by the robot's battery and memory. To control the robot you need to program him using the left panels.

Robot Moviments:
move forward, move backward, rotate left, rotate right

Control Operations:
repeat n times instruction
if condition instruction
if condition instruction else instruction

Conditions:
wall ahead, rock ahead, no wall ahead, no rock ahead
